# movie-list 🎥

simple list of movies and tv-shows from movie db

## requeriments:

- node (tested on version 10.13.0) ✔️️️
- web browser (tested on chrome 75) ✔️

## instructions:

- get all depencies
  `npm run install`
- start the project
  `npm run start` 😎

      maybe you need to open
      'http://localhost:3000'
      manually

      if you wanna deploy:

- build the project for deployment
  `npm run build`
